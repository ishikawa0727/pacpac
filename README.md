# pacpac

> **"生活の余白を、自らの手で上書きする。"**

わたしたちは、社会という大きな仕組みの中で共鳴し合うために、共通の「記号」や「スタイル」を必要としています。それは、日々を滑らかに進めるための大切な**潤滑油（API）**でもあります。

しかし、外部との効率的な対話を重視するあまり、私たちは時に、自らの内側から湧き上がる独自の視点を軽視し、記号的な枠組みに自分を委ねすぎてしまうことがあります。

500mlの牛乳パックは、その効率性と既視感という「共有された視点」の象徴です。

`pacpac`は、この完成された外部インターフェースとしてのパックを一度解体し、自らの内圧から生じた筆跡で意味付けし直すための装置です。

「外部との対話に必要な記号」と「自らの内側との対話」。
生活を構成するこのふたつの間の境界線を「わたし自身の視点」で上書きすること。

`pacpac`は、日々の生活の余白に、ささやかなひと筆を加えるための静かな実験室です。

---

## ✨ Key Features: 触覚を記述する技術

冒頭に掲げた哲学を実装するために、WebGPU を用いてデジタルな「手応え」を構築しています。

- **Visual Friction（視覚的摩擦）**: 
    紙と筆の間にあえて「抵抗」を介在させることで、ポインティングデバイスが身体の一部として溶けていくような感覚を追求します。
- **Fluid Dynamics & Translucency**: 
    WebGPU（Compute Shader）による粒子ベースの液体シミュレーション。粘性、色、透過率をカスタマイズし、パック内部に揺れる液体の質量を表現します。
- **Unfolding to Physical**: 
    デジタルで描いた質感と哲学を、現実世界へと展開図として持ち出す印刷機能を備えています。

## 🚀 Getting Started

```bash
# リポジトリをクローン
git clone git@github.com:ishikawa0727/pacpac.git

# ディレクトリに移動
cd pacpac

# 依存関係をインストール
npm install

# 開発サーバーを起動し、余白と向き合う
npm run dev
```
